# Trabalhos da Disciplina de Programação Orientada a Objetos

Requisitos:

- Desenhe o Diagrama de Classes completo do sistema.
  - Descrever todos os métodos gets e sets
  - Construtores default não é necessário
  - Os métodos de negócio devem ser descritos
  - Os construtores especializados devem ser descritos
  - A relação entre as classes é obrigatório
  - Deve ser persistido em arquivo as informações
- Implemente o sistema
- Será avaliado a solução do diagrama e a fidelidade com o sistema
- Uso correto da orientação a objetos
- Uso das funções, tecnologias e estratégias aprendidas para a solução
- A apresentação do sistema
- A solução definida
- Deverá ser apresentado em sala a solução
- Os requisitos questionados para o cliente devem ser registrados no programa (pergunta e a resposta)

## Tabalhos Individuais

### 1. Classificação de venda de imóvel (Real Estate Agency)

**Requisitos:**

1. Uma imobiliária quer registrar seus imóveis à venda. Para isso é necessário registrar se é casa, apto ou terreno.
2. Para casa registra-se nº quarto, nº banheiro, nº garagens, se possui mobília e a descrição de quais itens são acrescidos além do valor de venda.
3. Se for apto registra-se nº quarto, nº banheiro, nº garagens, se possui elevador, qual andar, se possui mobília e a descrição de quais itens são acrescidos além do valor de venda.
4. Se for terreno, se é de esquina, tamanho e o valor da venda.
5. Para todos os imóveis deve-se registrar o cliente com informação de CPF, Nome e data de nascimento.
6. Deve ser possível listar todos os dados dos imóveis por seu tipo.
